# クライアント納品手順（静的版）

## 渡すファイル

- `index.html`
- `styles.css`
- `app.js`
- `README.md`

上記4ファイルを同じフォルダのまま渡してください。

## クライアント側の使い方

1. `index.html` をブラウザで開く
2. イベントを追加・編集・削除する
3. 必要に応じて上部の `JSONを書き出し` で保存
4. 共有されたJSONがある場合は `JSONを読み込み`

## 仕様（共有について）

- サーバー不要で動作します
- データは各ブラウザの `localStorage` に保存されます
- 自動同期はしません（JSONファイルで受け渡し）

## 納品前チェック

- `index.html` を開いて表示される
- イベントの追加・編集・削除ができる
- `JSONを書き出し` でファイル保存できる
- `JSONを読み込み` で反映される
